# Curated Shader Examples Index

## Shadertoy — Elevated (IQ)
- **url**: https://www.shadertoy.com/view/MdX3Rr
- **techniques**: terrain, raymarching, fbm, fog, outdoor-lighting, atmosphere
- **domain**: glsl
- **summary**: IQ's iconic terrain shader. Raymarched FBM terrain with atmospheric fog, outdoor lighting, clouds. Demonstrates terrain marching optimization, height-based fog, and sky rendering. One of the most studied Shadertoy shaders.

## Shadertoy — Rainier (IQ)
- **url**: https://www.shadertoy.com/view/4ttSWf
- **techniques**: terrain, snow, atmospheric, volumetric-clouds, raymarching
- **domain**: glsl
- **summary**: Mountain landscape with volumetric clouds, snow coverage based on slope/height, and atmospheric scattering. Advanced terrain rendering with proper fog and sky integration.

## Shadertoy — Happy Jumping (IQ)
- **url**: https://www.shadertoy.com/view/3lsSzf
- **techniques**: sdf, animation, character, smooth-min, raymarching
- **domain**: sdf
- **summary**: Animated character made entirely from SDF primitives blended with smooth minimum. Demonstrates procedural animation, SDF composition, and organic shape creation from simple primitives.

## Shadertoy — Volcanic (IQ)
- **url**: https://www.shadertoy.com/view/XsX3RB
- **techniques**: terrain, lava, fbm, raymarching, emissive, glow
- **domain**: glsl
- **summary**: Volcanic terrain with glowing lava in crevices. FBM terrain with emissive coloring based on height/depth. Demonstrates combining terrain raymarching with emissive effects.

## Shadertoy — Clouds (IQ)
- **url**: https://www.shadertoy.com/view/XslGRr
- **techniques**: volumetric, clouds, raymarching, fbm, scattering
- **domain**: glsl
- **summary**: Volumetric cloud rendering using raymarched FBM noise. Beer-Lambert absorption, phase function for light scattering, and multiple noise octaves for cloud density. Foundation for understanding volumetric rendering.

## Shadertoy — Seascape
- **url**: https://www.shadertoy.com/view/Ms2SD1
- **techniques**: water, ocean, waves, raymarching, reflection, heightmap
- **domain**: glsl
- **summary**: Realistic ocean surface using layered sine waves for height, raymarched for intersection. Includes Fresnel reflection, specular highlights, and sky reflection. Popular reference for water rendering.

## Shadertoy — Protean Clouds
- **url**: https://www.shadertoy.com/view/3l23Rh
- **techniques**: volumetric, clouds, fbm, animated, colorful
- **domain**: glsl
- **summary**: Colorful animated volumetric clouds. Uses domain-warped FBM for cloud density with vibrant procedural color palette. Good study for combining noise, color, and volumetric techniques.

## Shadertoy — Fractal Land (Kali)
- **url**: https://www.shadertoy.com/view/XsBXWt
- **techniques**: fractal, kali-set, raymarching, abstract, procedural
- **domain**: glsl
- **summary**: Abstract fractal landscape using Kali-type iterative fractal. Demonstrates non-standard SDF fractals, orbit trap coloring, and creating otherworldly landscapes from simple iterative formulas.

## Three.js Journey — Shader Patterns
- **url**: https://threejs-journey.com/lessons/shader-patterns
- **techniques**: patterns, uv, step, smoothstep, sin, grid, circles, learning
- **domain**: glsl
- **summary**: 50+ shader patterns built from basic operations (step, smoothstep, sin, fract, mod). Excellent learning resource for understanding how to combine simple GLSL functions into complex visual patterns. Start here for shader fundamentals.

## The Book of Shaders — Noise
- **url**: https://thebookofshaders.com/11/
- **techniques**: noise, value-noise, random, hash, interpolation, learning
- **domain**: noise
- **summary**: Interactive introduction to noise in shaders. Builds from random hash → 1D noise → 2D value noise with smoothstep interpolation. Includes editable GLSL examples. Best beginner resource for understanding noise from first principles.

## The Book of Shaders — FBM
- **url**: https://thebookofshaders.com/13/
- **techniques**: fbm, fractal, octaves, domain-warping, learning
- **domain**: noise
- **summary**: Interactive FBM tutorial building on noise concepts. Demonstrates octave layering, lacunarity/gain parameters, turbulence (abs(noise)), ridge noise, and domain warping. Editable examples for experimentation.
